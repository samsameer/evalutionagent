## Recipe 1: Weekly Invoice Processing

**Goal:** Automatically process invoices every week and generate a summary.

**Setup:**
- Create folder: `Finance/Invoices/Incoming/`
- Create folder: `Finance/Invoices/Processed/`
- Create folder: `Finance/Reports/Weekly/`

**Steps:**

| Step | Action | Prompt |
|------|--------|--------|
| 1 | Scan for new invoices | "Check my Finance/Invoices/Incoming/ folder for any new PDF invoices" |
| 2 | Extract data | "Read each invoice and extract: vendor, date, amount, invoice number, and category" |
| 3 | Create spreadsheet | "Add all extracted data to Finance/Reports/Weekly/Invoices_Week_[date].xlsx" |
| 4 | Move processed files | "Move processed invoices to Finance/Invoices/Processed/[YYYY-MM]/" |
| 5 | Generate summary | "Create a summary showing total amount, count by category, and any invoices over $1000" |

**Combined Prompt:**
```
Check my Finance/Invoices/Incoming/ folder for new PDF invoices. For each one:
1. Extract the vendor name, date, amount, invoice number, and category
2. Add a row to Finance/Reports/Weekly/Invoices_Week_[today's date].xlsx
3. Move the processed invoice to Finance/Invoices/Processed/[YYYY-MM]/

After processing all invoices, add a summary sheet showing:
- Total amount for the week
- Count and total by category
- Any invoices over $1000 (flagged)

If there are no new invoices, create a brief note saying "No new invoices this week."
```

**Expected Output:** A weekly spreadsheet with all invoice data and a summary sheet.

## Recipe 2: Daily Market Research Monitor

**Goal:** Monitor specific markets daily and track changes.

**Setup:**
- Create folder: `Research/Daily-Monitor/`
- Create a master spreadsheet: `Research/Market_Tracker.xlsx`

**Combined Prompt:**
```
Perform my daily market research routine:

1. Search the web for new [property listings / product launches / industry news]
   related to [your specific criteria]

2. For each finding, record:
   - Title/Name
   - Source URL
   - Date found
   - Key details (price, location, features, etc.)
   - Your assessment (opportunity rating: High/Medium/Low)

3. Add new entries to my master tracker at Research/Market_Tracker.xlsx
   - Mark today's entries with today's date
   - Highlight any HIGH opportunity items

4. Save a daily summary to Research/Daily-Monitor/[today's date]_Summary.md

5. If any HIGH opportunity items were found, start the summary with
   "ACTION REQUIRED" and list them first
```

**Expected Output:** Updated master tracker + daily summary file.

## Recipe 3: Monthly Expense Report Generation

**Goal:** Compile all monthly expenses into a professional report.

**Setup:**
- Ensure receipts/invoices are in: `Finance/Expenses/[YYYY-MM]/`
- Create folder: `Finance/Reports/Monthly/`

**Combined Prompt:**
```
Generate my monthly expense report for [Month Year]:

1. Scan all files in Finance/Expenses/[YYYY-MM]/
2. Extract expense data from each receipt/invoice
3. Create Finance/Reports/Monthly/Expense_Report_[YYYY_MM].xlsx with:

   Sheet 1 - "Detail":
   - Date, Vendor, Description, Category, Amount, Payment Method
   - Sorted by date

   Sheet 2 - "Summary":
   - Total expenses for the month
   - Breakdown by category with subtotals
   - Top 10 largest expenses
   - Average daily spending

   Sheet 3 - "Comparison" (if previous month data exists):
   - Category-by-category comparison with previous month
   - Percentage change for each category
   - Notable increases or decreases (>20% change)

4. At the bottom of the Summary sheet, add:
   - "Report generated by Claude Cowork on [date]"
   - "Total expenses: $[amount]"
   - "Number of transactions: [count]"
```

**Expected Output:** Multi-sheet Excel workbook with detailed expense analysis.

## Recipe 5: Competitive Intelligence Automation

**Goal:** Regular competitive analysis with trend tracking.

**Setup:**
- Create folder: `Research/Competitors/`
- Create master file: `Research/Competitors/Competitor_Matrix.xlsx`

**Combined Prompt:**
```
Perform my competitive intelligence update:

Competitors to track:
- [Competitor 1]
- [Competitor 2]
- [Competitor 3]

For each competitor, research and update:
1. Current pricing (check their website)
2. Any new product/feature announcements (last 30 days)
3. Job postings (indicates growth areas)
4. Social media activity and customer sentiment
5. Any press mentions or news articles

Update Research/Competitors/Competitor_Matrix.xlsx with new findings.
Add a "Last Updated" column with today's date.

Create a brief report at Research/Competitors/Update_[date].md highlighting:
- Key changes since last update
- Potential threats or opportunities
- Recommended actions
```

**Expected Output:** Updated competitor matrix + analysis report.

## Recipe 6: End-of-Day File Cleanup

**Goal:** Automated daily cleanup of your working folders.

**Combined Prompt:**
```
Perform my end-of-day cleanup:

1. Downloads folder:
   - Move any PDFs to Documents/Unsorted/
   - Move any images to Photos/Unsorted/
   - Move any spreadsheets to Documents/Spreadsheets/
   - Leave everything else

2. Desktop:
   - If there are more than 10 files on my Desktop, move the oldest ones
     (by date modified) to Documents/Desktop-Archive/[today's date]/

3. Create a cleanup log at Cowork-Output/Logs/Cleanup_[date].txt listing:
   - Files moved and their new locations
   - Total files processed
   - Any files that couldn't be categorized (left in place)

Do NOT delete any files. Only move and organize.
```

**Expected Output:** Clean Downloads/Desktop + cleanup log.

## Tips for Creating Your Own Recipes

1. **Start with the end result** — What file or report do you want to see when the task is done?
2. **Break it into numbered steps** — Cowork follows numbered instructions well
3. **Specify exact paths** — Don't say "my folder," say "Finance/Invoices/"
4. **Include error handling** — "If you can't read a file, skip it and note it in the log"
5. **Add confirmation** — "Before deleting anything, list what you plan to remove and wait for my approval"
6. **Test with small batches** — Run the recipe on 5 files before unleashing it on 500
