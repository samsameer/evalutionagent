# Glossary

**Claude Cowork Mastery — March 2026**
**By Jabbir Basha, Principal AI Engineer**

---

Plain-language definitions for terms used throughout this course.

---

## A

**Agent** — An AI system that can take actions on your behalf, not just answer questions. Claude Cowork is an agent because it can read files, browse the web, and execute multi-step tasks.

**Anthropic** — The AI safety company that created Claude. Founded in 2021, Anthropic builds AI systems designed to be safe, beneficial, and understandable.

**API (Application Programming Interface)** — A way for software programs to communicate with each other. Developers use the Claude API to build applications powered by Claude. You don't need to know this for Cowork — it's for developers.

**Automation** — Having a computer perform tasks automatically without human intervention. In Cowork, this means setting up instructions that Claude follows repeatedly.

## C

**Claude** — Anthropic's AI assistant. Available in multiple versions (Haiku, Sonnet, Opus) with different speed and capability tradeoffs.

**Claude Code** — A command-line tool for developers that lets Claude write and edit code directly. Not required for this course.

**Claude Desktop** — The desktop application (macOS/Windows) required to use Cowork. This is where you interact with Claude and grant file access.

**Claude Pro** — Anthropic's paid subscription plan that includes access to Cowork features. The minimum subscription needed for this course.

**Cowork** — Claude's agent mode that can interact with your computer's files, browse the web, and execute multi-step tasks. Think of it as your AI co-worker.

**Context** — The information Claude has available when processing your request. This includes your conversation history, any files it has read, and the instructions you've given.

**Context Window** — The amount of text/information Claude can consider at once. Larger context windows mean Claude can work with bigger documents and longer conversations.

## D

**Data Extraction** — The process of pulling specific information out of documents (like extracting amounts from invoices or names from contracts).

## E

**Enterprise Plan** — Anthropic's business-grade subscription with additional security, admin controls, and team management features.

## F

**Folder Access** — The permission you grant Claude Cowork to read and write files in specific folders on your computer. You control exactly which folders Claude can access.

## H

**Haiku** — The fastest and most lightweight Claude model. Best for simple, quick tasks. Part of the Claude model family.

## L

**LLM (Large Language Model)** — The type of AI technology that powers Claude. LLMs are trained on vast amounts of text and can understand and generate human language.

## M

**MCP (Model Context Protocol)** — An open protocol created by Anthropic that allows AI models to connect with external tools and data sources. It enables Claude to interact with databases, APIs, and other services.

**Multi-Step Workflow** — A task that requires multiple actions in sequence. For example: read a PDF → extract data → create a spreadsheet → generate a report.

## O

**Opus** — The most capable and intelligent Claude model. Best for complex analysis, nuanced writing, and difficult tasks. Part of the Claude model family.

## P

**Permission** — The access rights you give Claude Cowork. You decide which folders it can read, which it can write to, and what actions require your confirmation.

**Prompt** — The instruction or question you give to Claude. Better prompts lead to better results. This course teaches you how to write effective prompts.

## R

**Recurring Task** — An instruction that Cowork executes on a schedule (daily, weekly, monthly). Example: "Every Monday, organize my Downloads folder."

## S

**SDK (Software Development Kit)** — A toolkit for developers to build applications. Anthropic provides SDKs for Python and TypeScript. Not required for this course.

**Skills** — Predefined capabilities that Claude can execute. Skills include file management, data analysis, web research, and more.

**Sonnet** — The balanced Claude model offering a good mix of speed and capability. Part of the Claude model family.

## T

**Team Plan** — Anthropic's subscription plan designed for teams and organizations, with collaboration features and shared billing.

**Token** — The unit of text that AI models process. Roughly, 1 token equals about 3/4 of a word. Subscription plans include token allowances.

## U

**Unstructured Data** — Information that isn't organized in a predefined format, like PDFs, emails, or handwritten notes. Cowork excels at turning unstructured data into structured formats (like spreadsheets).

## W

**Workflow** — A series of steps that accomplish a specific business task. Cowork automates workflows that would otherwise require manual effort.
