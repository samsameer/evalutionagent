## Module Overview

GitHub Copilot CLI brings AI assistance directly to your terminal. Instead of Googling complex shell commands, you can ask Copilot to explain commands you do not understand or suggest the right command for any task. It works with shell commands, Git operations, and the GitHub CLI.

---

## Learning Objectives

- Install and authenticate the Copilot CLI extension
- Use `gh copilot explain` to understand complex commands
- Use `gh copilot suggest` to generate commands for any task
- Build advanced terminal workflows with Copilot assistance
