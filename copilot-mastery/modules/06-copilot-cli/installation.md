## Quick Setup Summary

```bash
# 1. Install GitHub CLI (if not already installed)
brew install gh        # macOS
winget install GitHub.cli  # Windows
sudo apt install gh    # Ubuntu/Debian

# 2. Authenticate
gh auth login

# 3. Install Copilot extension
gh extension install github/gh-copilot

# 4. Verify
gh copilot --version
```
