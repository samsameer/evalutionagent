## Module Overview

Before you can experience the power of GitHub Copilot, you need to install and configure it in your development environment. This module walks you through setting up Copilot in every supported editor — VS Code, JetBrains IDEs, Neovim, Xcode — and the CLI.

By the end of this module, you will have Copilot running in your editor of choice with code completions and chat fully functional.

---

## Learning Objectives

- Install the GitHub Copilot extension in VS Code
- Set up Copilot in JetBrains IDEs (IntelliJ, PyCharm, WebStorm, etc.)
- Configure Copilot for Neovim and Xcode
- Install and authenticate the Copilot CLI
- Verify that Copilot is active and generating suggestions
