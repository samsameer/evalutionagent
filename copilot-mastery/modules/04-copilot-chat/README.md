## Module Overview

Copilot Chat is where the magic of conversational AI meets your code editor. Instead of just accepting suggestions line by line, you can have rich, contextual conversations about your code — ask questions, request refactors, generate tests, fix bugs, and much more.

This module covers every aspect of Copilot Chat including the different chat modes, all chat participants, every slash command, context variables, and even vision capabilities for working with images.

---

## Learning Objectives

- Master all three chat modes: Panel, Inline, and Quick Chat
- Use chat participants (@workspace, @terminal, @vscode, @github) effectively
- Leverage slash commands (/explain, /fix, /tests, /doc, /new, /optimize)
- Reference specific context with variables (#file, #selection, #editor, #codebase)
- Use vision capabilities to ask about screenshots and diagrams
