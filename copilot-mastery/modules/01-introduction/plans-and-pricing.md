## Overview

## Understanding Premium Requests

Starting in 2025, GitHub introduced the concept of **premium requests** — these are interactions that use more expensive, advanced models or agentic features.

### What Counts as a Premium Request?

- Using **o1** or **o3-mini** models in Copilot Chat
- Using the **Copilot Coding Agent** to resolve issues
- Certain complex **agent mode** operations
- **Copilot Workspace** sessions

### What Does NOT Count as a Premium Request?

- Standard code completions (GPT-4o)
- Regular Copilot Chat with GPT-4o or Claude 3.5 Sonnet
- Copilot Edits with standard models
- Copilot CLI usage

### Premium Request Allowances

| Plan | Monthly Premium Requests | Overage |
|------|-------------------------|---------|
| Free | 0 | Not available |
| Pro | 0 | Not available |
| Pro+ | 1,500 | $0.04/request |
| Business | 0 | Not available |
| Enterprise | 1,000/user | $0.04/request |

## Which Plan Should You Choose?

### Choose **Free** if:
- You are a student or hobbyist
- You want to try Copilot before committing
- You write code occasionally and 2,000 completions/month is enough
- You do not need CLI access or advanced features

### Choose **Pro** ($10/month) if:
- You are a professional developer coding daily
- You want unlimited completions and chat messages
- You need multi-model selection
- You want Copilot CLI, agent mode, and NES
- You do not need the Copilot Coding Agent

### Choose **Pro+** ($39/month) if:
- You want access to the **Copilot Coding Agent** (assign issues to Copilot)
- You need premium models (o1, o3-mini) with 1,500 premium requests
- You want full Copilot Workspace access
- You are a power user who wants every cutting-edge feature

### Choose **Business** ($19/user/month) if:
- You are a team or organization
- You need **IP indemnity** for legal protection
- You require content exclusions and organization-wide policy controls
- You need audit logs for compliance
- You want centralized billing and seat management

### Choose **Enterprise** ($39/user/month) if:
- You are a large organization
- You need everything in Business PLUS:
  - Knowledge bases for custom documentation
  - SAML SSO integration
  - Premium requests (1,000/user/month)
  - Copilot Coding Agent access
  - Full Copilot Workspace access
  - Priority support

## Copilot for Students and Educators

GitHub provides **free Copilot Pro access** to verified students and educators through the [GitHub Education](https://education.github.com) program. If you have a valid `.edu` email address, you can get:

- Free Copilot Pro plan
- Free GitHub Pro account
- Access to the GitHub Student Developer Pack

---

## Copilot for Open Source Maintainers

Verified open source maintainers can also get **free Copilot access**. If you maintain a popular open source project on GitHub, check your Copilot settings — you may already be eligible.

---

## How to Subscribe

1. Go to [github.com/settings/copilot](https://github.com/settings/copilot)
2. Select your desired plan
3. Complete the payment process
4. Copilot will be immediately available in your supported editors
