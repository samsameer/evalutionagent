## Module Overview

This is where GitHub Copilot evolves from a suggestion tool into an **autonomous coding agent**. Copilot Edits lets you make AI-driven changes across multiple files, while Agent Mode takes it further — autonomously planning, coding, running commands, and iterating until the task is complete.

These are the most powerful features in the entire Copilot ecosystem, and mastering them will fundamentally change how you write software.

---

## Learning Objectives

- Use Copilot Edits for multi-file editing sessions with diff review
- Master Agent Mode for autonomous multi-step task execution
- Understand how agent mode runs terminal commands and self-corrects
- Configure MCP servers to give agent mode access to external tools
