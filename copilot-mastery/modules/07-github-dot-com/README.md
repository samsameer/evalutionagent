## Module Overview

GitHub Copilot is not just for your editor — it is deeply integrated into **GitHub.com** itself. You can chat with Copilot about any repository, generate AI-powered PR summaries, get AI code reviews, and use Copilot Workspace to go from an issue to a complete pull request.
