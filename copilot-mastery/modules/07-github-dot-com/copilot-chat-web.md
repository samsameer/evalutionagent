## Overview

Copilot Chat is available directly on **github.com**, allowing you to ask questions about any repository, get code explanations, and navigate codebases — all without leaving your browser.

---

## Accessing Copilot Chat on GitHub.com

### Method 1: Global Chat Icon

1. Look for the **Copilot icon** in the top-right corner of any GitHub page
2. Click it to open a chat panel
3. Start asking questions

### Method 2: Repository-Specific Chat

1. Navigate to any repository
2. Click the **Copilot icon** or find it in the repository navigation
3. Your questions will be scoped to that repository's codebase

### Method 3: Keyboard Shortcut

Press **`/`** (forward slash) on any GitHub page to focus the search bar, then look for the Copilot option.

## What You Can Ask

### Repository Questions

```
How is the authentication system implemented in this repo?
What testing framework is used?
Where are the API routes defined?
Explain the database schema used in this project
What CI/CD pipeline is configured?
```

### Code Understanding

```
What does the function processPayment do in src/services/payment.ts?
Explain the algorithm in utils/sorting.go
Why does this project use Redis alongside PostgreSQL?
```

### Navigation

```
Where is the main entry point of this application?
Which files handle user registration?
Show me the error handling middleware
```

### General Programming

```
What's the difference between useMemo and useCallback in React?
How do I implement a binary search tree in Python?
What are the SOLID principles?
```

## Context-Aware Conversations

When you are viewing a specific file, pull request, or issue on GitHub.com, Copilot Chat automatically has context about what you are looking at:

- **On a file page:** Copilot can answer questions about that specific file
- **On a PR page:** Copilot understands the changes in the pull request
- **On an issue page:** Copilot can discuss the issue and suggest solutions
- **On a commit page:** Copilot can explain the changes in that commit

## GitHub.com Chat vs. VS Code Chat

| Feature | GitHub.com | VS Code |
|---------|------------|---------|
| Ask about code | Yes | Yes |
| Repo-wide context | Yes | Yes (@workspace) |
| Edit code | No | Yes |
| Agent mode | No | Yes |
| Copilot Edits | No | Yes |
| Run commands | No | Yes |
| Model selection | Yes | Yes |
| Chat history | Session-based | Session-based |
| Offline | No | No |
