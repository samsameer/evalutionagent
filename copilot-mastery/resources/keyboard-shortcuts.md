# Keyboard Shortcuts Reference

**GitHub Copilot Mastery — March 2026**

## VS Code — Code Completions

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Accept full suggestion | `Tab` | `Tab` |
| Accept next word | `Ctrl+Right Arrow` | `Cmd+Right Arrow` |
| Accept next line | `Ctrl+Right Arrow` (at EOL) | `Cmd+Right Arrow` (at EOL) |
| Dismiss suggestion | `Escape` | `Escape` |
| Show next suggestion | `Alt+]` | `Option+]` |
| Show previous suggestion | `Alt+[` | `Option+[` |
| Trigger suggestion manually | `Alt+\` | `Option+\` |
| Open completions panel | `Ctrl+Shift+Alt+]` | — |

---

## VS Code — Copilot Chat

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Open Chat Panel | `Ctrl+Alt+I` | `Cmd+Option+I` |
| Open Inline Chat | `Ctrl+I` | `Cmd+I` |
| Open Quick Chat | `Ctrl+Shift+I` | `Cmd+Shift+I` |
| Accept inline chat change | `Ctrl+Enter` | `Cmd+Enter` |
| Discard inline chat change | `Escape` | `Escape` |

---

## VS Code — Next Edit Suggestions (NES)

| Action | Windows/Linux | macOS |
|--------|--------------|-------|
| Accept NES suggestion | `Tab` | `Tab` |
| Jump to next NES | `Tab` (after accept) | `Tab` (after accept) |
| Dismiss NES | `Escape` | `Escape` |

## Neovim

| Action | Keybinding |
|--------|-----------|
| Accept suggestion | `Tab` |
| Dismiss suggestion | `Ctrl+]` |
| Next suggestion | `Alt+]` |
| Previous suggestion | `Alt+[` |
| Trigger suggestion | `:Copilot suggest` |
| Open panel | `:Copilot panel` |
| Check status | `:Copilot status` |
