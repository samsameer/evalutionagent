# Module 05 — Web Research Agents

**Claude Cowork Mastery — March 2026**
**Instructor:** Jabbir Basha, Principal AI Engineer

---

## Overview

One of Claude Cowork's most powerful capabilities is its ability to browse the web, conduct research, and deliver summarized findings directly to you. In this module, you will learn how to configure web browsing, set up research workflows, and receive polished, summarized reports — all without leaving Cowork.

Whether you need to monitor competitors, track market trends, or gather pricing data, Cowork can handle the heavy lifting while you focus on making decisions.

---

## Learning Objectives

By the end of this module, you will be able to:

- **Configure web browsing** — Enable and customize Cowork's web access settings for research tasks.
- **Set up research workflows** — Design repeatable research processes that Cowork can execute on demand.
- **Get summarized reports** — Receive organized, actionable reports from Cowork's web research in the format you prefer.
