# Getting Started — Installation & Setup

**Module 01, Lesson 1 | Claude Cowork Mastery — March 2026**
**Instructor:** Jabbir Basha, Principal AI Engineer

---

## Overview

In this lesson, you'll download the Claude Desktop application, sign in with your paid subscription, and verify that Cowork is ready to use.

---

## Step 1: Get a Claude Pro Subscription

Before downloading anything, ensure you have an active paid subscription:

1. Visit [claude.ai](https://claude.ai)
2. Click **Sign Up** or **Log In**
3. Navigate to **Settings → Subscription**
4. Subscribe to **Claude Pro**, **Team**, or **Enterprise**
5. Confirm your payment and subscription is active

> **Remember:** The free tier does NOT include Cowork access.

---

## Step 2: Download Claude Desktop

The Claude Desktop app is required for Cowork — it cannot run in the web browser alone.

### For macOS:
1. Visit the [Anthropic Downloads page](https://claude.ai/download)
2. Click **Download for macOS**
3. Open the downloaded `.dmg` file
4. Drag **Claude** into your **Applications** folder
5. Open Claude from Applications

### For Windows:
1. Visit the [Anthropic Downloads page](https://claude.ai/download)
2. Click **Download for Windows**
3. Run the downloaded `.exe` installer
4. Follow the installation wizard
5. Launch Claude from the Start Menu or Desktop shortcut

---

## Step 3: Sign In & Activate Cowork

1. Open the Claude Desktop application
2. Sign in with your Anthropic account (the one with the paid subscription)
3. Look for **Cowork** in the navigation panel or sidebar
4. If you see the Cowork option, you're all set!

### What If You Don't See Cowork?

- **Check your subscription:** Cowork requires Pro, Team, or Enterprise
- **Update the app:** Ensure you have the latest version of Claude Desktop
- **Check availability:** Cowork is being rolled out gradually — check Anthropic's status page
- **Restart the app:** Sometimes a fresh restart activates new features

---

## Step 4: Explore the Interface

Take a few minutes to familiarize yourself with the Claude Desktop interface:

- **Chat Panel:** Where you type instructions and have conversations
- **Cowork Panel:** Where agent tasks are managed and monitored
- **File Access Indicator:** Shows which folders Claude can access
- **Settings Gear:** Configure permissions, folders, and preferences

---

## Verification Checklist

Before moving to the next lesson, confirm:

- [ ] Claude Desktop is installed and running
- [ ] You're signed in with a paid subscription
- [ ] You can see the Cowork option in the app
- [ ] You've explored the basic interface layout

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| App won't install | Check system requirements (macOS 12+ or Windows 10+) |
| Can't sign in | Verify your email and password at claude.ai first |
| Cowork not visible | Update the app to the latest version |
| Subscription not recognized | Sign out and sign back in |
