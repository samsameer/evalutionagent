## Module Introduction

Welcome to Module 02 of Claude Cowork Mastery. In this module, you will build the foundational skills needed to work effectively with Claude Cowork. We start by setting up your Desktop application, move on to granting Claude safe access to your project folders, and finish by having your very first productive conversation with Cowork.

By the end of this module, you will have a fully configured environment and the confidence to start delegating real tasks to Claude.

---

## Learning Objectives

After completing this module, you will be able to:

1. **Navigate the Claude Desktop interface** — Understand every panel, menu, and setting available to you.
2. **Configure Cowork preferences** — Tailor the agent experience to suit your workflow and comfort level.
3. **Enable agent features** — Turn on file operations, command execution, and other Cowork capabilities.
4. **Grant folder access securely** — Give Claude the file access it needs while keeping sensitive data protected.
5. **Conduct your first Cowork conversation** — Write clear instructions, understand how Claude responds, and handle action confirmations.
6. **Avoid common beginner mistakes** that slow down new users.
