## Module Overview

Welcome to Module 06! In this module, you will learn how to put Claude Cowork on autopilot by creating scheduled automations. Instead of giving instructions one at a time, you can set up recurring tasks and multi-step workflows that run on a schedule — freeing you to focus on higher-value work while Cowork handles the routine.

By the end of this module, you will be able to automate repetitive processes, chain tasks into powerful workflows, and confidently monitor the results to ensure everything runs smoothly.

---

## Learning Objectives

By completing this module, you will be able to:

- **Create recurring instructions** that run on a daily, weekly, monthly, or custom schedule
- **Build multi-step workflows** that chain several tasks together into a single automation
- **Monitor and review outputs** to ensure your automations are producing accurate, high-quality results
