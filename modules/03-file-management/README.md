## Module Overview

Managing files manually is tedious, error-prone, and time-consuming. In this module, you will learn how to use Claude Cowork to automate your file management workflows — from sorting messy folders to batch renaming documents and setting up safe backup strategies.

By the end of this module, you will be able to hand off your most repetitive file tasks to Cowork with confidence.

---

## Learning Objectives

By completing this module, you will be able to:

- **Auto-sort files** into organized folders based on type, date, project, or client
- **Batch rename files** using consistent naming conventions with a single prompt
- **Organize messy folders** into clean, structured directory layouts
- **Create safe backup workflows** to protect your data before making automated changes
