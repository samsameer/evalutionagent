# Module 11: Cheat Sheets & Quick Reference

**Module 11, Overview — Claude Code Mastery**

**Instructor: Jabbir Basha, Principal AI Engineer**

---

## Overview

This module is your one-stop quick reference for everything Claude Code. Rather than teaching new concepts, it consolidates the entire course into printable, searchable reference materials. Keep these files open in a second terminal, pin them to your browser, or print them out. When you forget a flag, a shortcut, or why Claude is not behaving as expected, this module has the answer in seconds.

## Prerequisites

- Completed Modules 01-10 (or at least skimmed them)
- A working Claude Code installation
- Familiarity with the core concepts covered in earlier modules

---

## Learning Objectives

By the end of this module, you will be able to:

1. **Look up any command, flag, or setting** without leaving your editor
2. **Navigate Claude Code efficiently** using keyboard shortcuts
3. **Write effective prompts** that produce accurate, first-try results
4. **Diagnose and fix common issues** using a structured troubleshooting approach
5. **Print reference cards** to keep at your desk for quick access

## How to Use This Module

- **command-reference.md** is the master cheat sheet. Bookmark it. Search it with Ctrl+F.
- **keyboard-shortcuts-reference.md** is designed to be printed. The ASCII art format survives any printer.
- **prompt-engineering.md** is for when you know what you want Claude to do but cannot get it to do it right.
- **troubleshooting-guide.md** is for when something is broken. Start at the top and follow the decision tree.
