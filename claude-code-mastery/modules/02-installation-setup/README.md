## Module Overview

Before you can start using Claude Code, you need to install it, authenticate, and configure your terminal. This module walks you through every step — from `npm install` to your first productive session.

---

## Learning Objectives

- Install Claude Code on macOS, Linux, and Windows (WSL)
- Authenticate with your Anthropic account or API key
- Configure your terminal for the best experience
- Complete your first interactive session successfully

## Prerequisites

Before starting, make sure you have:

- **Node.js 18+** — Check with `node --version`
- **npm** — Comes with Node.js, check with `npm --version`
- **Git** — Check with `git --version`
- **A modern terminal** — iTerm2, Ghostty, WezTerm, Kitty, VS Code terminal, or similar
- **An Anthropic account** — Claude Pro, Max, Team, Enterprise, or API key
