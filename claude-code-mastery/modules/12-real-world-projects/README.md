# Module 12: Real-World Projects

**Module 12, Overview — Claude Code Mastery**

**Instructor: Jabbir Basha, Principal AI Engineer**

---

## Overview

This module brings together everything you have learned across the course by applying Claude Code to real-world development scenarios. Instead of isolated exercises, you will build a full-stack application from scratch, set up an automated code review pipeline, configure CI/CD automation, and contribute to open source projects — all guided by Claude Code. Each lesson provides the exact prompts and workflows you would use on a production project.

## Prerequisites

- Completed Modules 01-11 (especially tools, hooks, headless mode, and team features)
- A working Claude Code installation with API or Pro plan access
- Node.js 18+ and PostgreSQL installed (for Lesson 1)
- A GitHub account with SSH keys configured
- Familiarity with Git workflows and pull requests

---

## Learning Objectives

By the end of this module, you will be able to:

1. **Build a full-stack application** end-to-end using Claude Code as your development partner
2. **Set up automated code review** that runs on every pull request with custom rules
3. **Integrate Claude Code into CI/CD pipelines** for testing, linting, and security scanning
4. **Contribute to open source projects** efficiently by using Claude Code to understand unfamiliar codebases
5. **Apply production-grade workflows** that combine multiple Claude Code features together

## Module Structure

Each lesson is a self-contained project you can complete independently:

- **Lesson 1** walks through building a complete application, showing how to prompt Claude Code at each stage
- **Lesson 2** sets up code review automation that catches issues before they reach production
- **Lesson 3** integrates Claude Code into your deployment pipeline for continuous quality
- **Lesson 4** demonstrates an effective workflow for contributing to projects you did not write
