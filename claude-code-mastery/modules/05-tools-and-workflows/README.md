## Module Overview

Claude Code comes with a powerful suite of **built-in tools** that handle everything from reading files and editing code to searching the web and managing Git workflows. Understanding these tools — what each one does, when to use it, and how they compose together — is what separates a casual user from a power user.

This module is your **definitive reference** for every tool and workflow pattern available in Claude Code.

---

## Learning Objectives

- Know every built-in tool and its parameters
- Master file editing workflows using Edit, Write, and Read
- Run Git operations — commits, branches, PRs, and worktrees — entirely through Claude Code
- Use web tools to fetch documentation, search for solutions, and pipe content into analysis
- Combine tools into efficient multi-step workflows

## Key Concepts

Claude Code does not execute your requests by generating a single monolithic script. Instead, it **orchestrates a set of discrete tools**, calling them in sequence or in parallel depending on the task. Each tool is purpose-built:

| Category | Tools | Purpose |
|----------|-------|---------|
| File I/O | Read, Edit, Write | Read, modify, and create files |
| Search | Glob, Grep | Find files and search content |
| Execution | Bash | Run shell commands |
| Web | WebSearch, WebFetch | Search the internet and fetch URLs |
| Orchestration | Agent, TodoWrite | Parallel sub-tasks and progress tracking |
| Notebooks | NotebookEdit | Edit Jupyter notebook cells |

Understanding which tool to reach for — and which to avoid — is the core skill this module teaches.
